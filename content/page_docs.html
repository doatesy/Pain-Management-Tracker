About

This app was intended to be a simple way for an individual to track points of pain in their body and generate reports from the data that is entered. There was no professional assistance with the design or coding, so if you do know what you are doing and have some suggestions, I would love to hear them via the github page.

Usage of the app should be fairly straight forward. Download the app, open the index file, create a user, and click on the body image to add a new pain point, or click and existing point to edit it (extend timeframe, change details, etc).

I designed this as a pure HTML5 app utilising local browser storage so that it should (in theory) be able to run on any device without the need to install software. The downside to storing data in the HTML5 local storage is that it can be easily wiped when clearing browser temporary files, being slightly difficult to manipulate when trying to deal with database records and fields, as well as being unencrypted.

Getting Started

Download

First thing to do is to download the app. If you are reading this though, chances are you have already. The app can be downloaded from the github page. Instructions should be on this page on how to do this.

First Run

  1. Open the index.html file from your download in a modern browser
  2. You should be presented with a page like the on on the right
  3. Type your name in the "name" field & click "submit"
  4. You should now see your name in the list of "Existing Users"
  5. Click on your name. You should be taken to the "Pain Management" page with an outline of the human body.
Note : The app will remember the last user selected, and the last page you were on, and automatically load them when you open the page next time. If you want to log out, change user, or add a new user, click the "Change User" button up the top.

Logging pain/entering pain points

  1. You can click on any part body outline to create a new entry, or to edit an existing point, just click on it.
  2. You should see the "Pain Details" page like the one on the right.
  3. Enter the details of the pain you are having into the fields listed. A brief description of each field/option should be displayed.
  4. You can use the preset buttons in the pain duration section. Please note that you must use the format supplied (eg YYYY-MM-DD HH:MM:SS)
  5. Notes are searchable, so you can store some additional notes/keywords in here and search for them later.
  6. Once finished, click the "Save" button. The close button will close the window without saving the pain details.
  7. You should now see the human body outline with your new pain point added on it. You can move your mouse over the top of it to see some quick details.
Note : If are adding a new pain point and you realised you clicked in the wrong spot, just close the Pain Details window and click the correct spot. The old spot will not be saved.
